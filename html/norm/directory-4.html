CSS规范

css书写规范

  1. css类命名参考内容命名

  2. 禁止使用层级过深的选择器，最多3级

    错误示例：

    			
				
.main .container .wrap .box .blue-color{}
.wrap .container .box .red-color{}
  3. 除非有特定的功能、组件要求等，禁止随意使用id来定义元素样式

  4. 除非是样式reset需要，禁止对纯元素选择器设置特定样式，避免样式污染

    错误示例：

    			
				
// 会导致页面所有的a标签都被加上背景
a{background:url(XXX);}

//后期修改可能会添加一些span标签，如果刚好在div里面，会被污染；不利于后期维护
div span{display:block}
  5. 尽量少用!important
  6. 少用行内样式sytle=""
  7. 尽量使用属性缩写写法

  8. 尽量缩写16进制颜色代码

    示例：

    			
				
color:#000; // #000000
color:#fed; // #ffeedd

  9. 去掉小数点前的“0”

    示例：

    			
				
font-size:.3rem
font-size.2em

CSS书写顺序

正确的css书写顺序不仅易于查看，并且也属于css样式优化的一种方式

  1. 定位属性：position, top, right, z-index, display, float等
  2. 盒子属性：width, height, padding, margin
  3. 文字系列：font, line-height, letter-spacing, color等
  4. 背景：background, border等
  5. 其他：animation, transition等

示例：

	
		
// 此示例只是用来展示样式书写顺序
.box{
	position:absolute;
	top:0;
	right:0;
	bottom:0;
	left:0;
	z-index:99;
	display:flex;
	width:100px;
	height:100px;
	padding:10px;
	margin:10px;
	line-height:80px;
	font-size:16px;
	color:#666;
	background:#000;
	border:1px solid #999;
	border-radius:5px;
	.
	.
	.
}
